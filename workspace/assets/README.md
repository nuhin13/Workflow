# workspace/assets/ — project evidence and imports

**Why this exists.** Pipeline steps sometimes need supporting files: forecast
data, screenshots, exported design assets, or imported prototype files.

| Folder | Purpose |
|---|---|
| `forecast/` | Forecast inputs and cited market/evidence files |
| `design-imports/` | Figma/Stitch/prototype exports used by `/design` |

**What it does NOT cover.** Source code belongs in `src/`; generated plans live
in `workspace/plan/`.
