# workspace/docs/ — product source documents

**Why this exists.** These are the human-approved product inputs that drive the
pipeline.

| Folder | Purpose |
|---|---|
| `business/` | Approved BRD and business context; required before kickoff |
| `design/` | Approved UI reference; required before kickoff, Figma is law when linked |

**What it does NOT cover.** Generated phase artifacts live in
`workspace/plan/`; the canonical build spec lives in `workspace/spec/`.
