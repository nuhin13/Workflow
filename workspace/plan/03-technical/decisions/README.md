# Product Architecture Decisions

`/tech-plan` creates accepted product decisions here as
`ADR-NNNN-<slug>.md`, starting at `ADR-0001`.

Agents may propose options and recommend one, but the human owns foundational
choices. Accepted records are immutable; changes use a new ADR that
supersedes the old one. The template is `harness/templates/plan/adr.md`.

Harness-engine decisions use the separate `HADR-NNNN` namespace under
`harness/memory/decisions/`.
