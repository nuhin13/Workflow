# Context Pruning
- After each checklist item: write a 3–5 line note (what changed, where,
  result) and mentally drop the raw diffs/file bodies behind it.
- Keep permanently: the task file, current errors, your notes, open
  questions. Everything else is re-fetchable.
- Long sessions: compact at natural checkpoints (post-commit) — summarize
  "state so far" in ≤10 lines, continue from the summary.
- Failed approaches: record the one-line reason it failed, prune the rest —
  the lesson, not the transcript.
